Resumen del proyecto:

El proyecto consiste en un sistema que permita almacenar y gestionar la información de una empresa encargada de asesorar y capacitar en prevención de riesgos.
Se pide, por lo tanto, gestionar la información relacionada a la gestión de recursos humanos. Se ha solicitado la creación de un proyecto Web Dinámico, utilizando
el patrón de diseño MVC utilizando Spring Framework junto con Spring Security,para el manejo de las sesiones de usuario, basados en los siguientes puntos principales:

1- Login.
2- Creación de usuarios.
3.-Asignación tipo usuario.
4.-Listados de usuarios.
5.-Creación de capacitación.
6.-Listados de capacotación.
7.-Edición de los distintos tipos de usuario.
8.-Creación de visitas.
9.-Listados de visitas.
10.-Creación de pagos.
11.-Listados de pagos.
12.-Responder checklist.
13.-Asignar checklist.

Durante el desarrollo del sistema, se utilizaron conceptos y fundamentos pertenecientes a:
-	HTML.
-	CSS. 
-	Java Script.
-	Responsive.
-	GitHub.
- 	Controladores.
-	Jsp.
-	Tomcat.
-	Base de datos(ORACLE).
-       JPA.
-	Spring Framework.
-	Spring Security.

Todos los códigos presentes fueron codificados mediante el entorno de desarrollo Eclipse y almacenados en un repositorio en Github.
